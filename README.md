### 🌐 What I'm Building: World Credit

**[World Credit](https://world-credit.com)** is a universal trust score — a single, portable reputation that proves you're real, reliable, and trustworthy.

We aggregate signals from across your digital life — identity verification, financial accounts, social networks, professional profiles, and peer vouches — into one score (0–100, Bronze through Platinum) that travels with you. No credit pulls. You control what you share.

**I built the entire platform solo:**

| Component | Stack |
|:--|:--|
| 📱 App (iOS & Android) | Flutter, Dart, BLoC, Firebase Auth (Apple, Google, Phone) |
| 🔥 Backend | Firebase, Cloud Functions (TypeScript), Firestore |
| 💳 Billing | Stripe (subscriptions, checkout, webhooks) |
| 🌐 Website | Static HTML/CSS, Firebase Hosting |
| 📦 SDKs | Web (JS), iOS (Swift Package), Android (Kotlin), Flutter (Dart) |
| 🔑 Badge API | Cloud Functions, API key auth, tiered rate limiting |
| 🛡️ Dashboard | Stripe-integrated developer portal for API key management |

**For platforms** — Embed trust badges (inline, pill, card, shield) directly into your product with drop-in SDKs for Web, iOS, Android, and Flutter. Or call the REST API directly. Reduce fraud, increase trust, and make better decisions about your users.

### 🌿 What I'm Building: Skape

**[Skape](https://skape.us)** is an on-demand landscaping marketplace — think Uber, but for lawn care.

Homeowners select their property, pick services, and book instantly. Landscapers bid on jobs, manage their schedule, and get paid through the app. No phone calls, no quotes, no back-and-forth.

**I built the entire platform solo:**

| Component | Stack |
|:--|:--|
| 📱 Homeowner App (iOS & Android) | Flutter, Dart, BLoC, Google Maps SDK |
| 🌿 Landscaper App (iOS & Android) | Flutter, Dart, BLoC, Google Maps SDK |
| 🧠 AI Property Analysis | GPT-4o, SAM 2, Modal (GPU), ArcGIS/State GIS APIs |
| 🔥 Backend | Firebase, Cloud Functions (TypeScript), Firestore |
| 💳 Payments | Stripe Connect (payouts, connected accounts) |
| 🔍 Search | Algolia (full-text + geosearch) |
| 🌐 Website | Next.js, Tailwind CSS |
| 🛡️ Admin Dashboard | Next.js, Tailwind, Mapbox GL, Firebase client SDK |
| ⚙️ CI/CD | GitHub Actions, Fastlane |

**AI Property Analysis** — Homeowners enter their address, and the app automatically segments their property from satellite imagery. It identifies the house, driveway, and lawn using a pipeline of public GIS parcel data, Google Maps satellite imagery, Meta's SAM 2 model for image segmentation (running on Modal GPUs), and GPT-4o for feature classification. The lawn boundary comes directly from government parcel data — no AI guessing. The result is real lat/lng polygons displayed on an interactive Google Map.

> **10 states** · **19 service areas** · **Registered U.S. Trademark** · [Skape-Industries on GitHub](https://github.com/Skape-Industries)

### How I Think About Building Things

- **Ship, then iterate.** Perfect is the enemy of live. I'd rather have real users finding real bugs than a flawless prototype nobody uses.
- **Architecture pays for itself.** I write RFCs before code, use ADRs to document decisions, and practice trunk-based development. It's slower on day one and faster every day after.
- **Make others faster.** Leading isn't about writing the most code — it's about unblocking 10 engineers so the whole team ships faster.
- **Own the full stack.** I don't throw things over the wall. If I build the frontend, I own the backend, the deployment, the monitoring, and the on-call.
